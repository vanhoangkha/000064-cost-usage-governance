#### Create a policy ( policy ) that limits service usage by EC2 instance size.

Amazon Elastic Compute Cloud (EC2) virtual server service provides us with many different configuration options according to family and instance size. In this section we will proceed to create a policy that only allows the creation and use of EC2 instances with a T series family and an instance size of **t3.nano**.
